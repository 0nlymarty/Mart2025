Git ignore

Hva er .gitignore?

.gitignore-filen brukes til å spesifisere hvilke filer og kataloger Git skal ignorere i et prosjekt.
Dette er nyttig for å unngå å inkludere midlertidige filer, konfigurasjonsfiler eller andre filer som ikke skal versjonskontrolleres.

Eksempel på .gitignore

*.txt *.css *.jpg *.html *.py osv

Hvordan fungerer dette? Vel det fungerer fordi gitignore forteller git hvilke fil typer og mapper den skal ignorere,



det betyr at git ikk elenger sporer, leger til eller commiter de filene
