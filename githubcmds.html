Git kommandoer

Her er noen vanlige git kommandoer du kan bruke i terminalen

git init - lager et nytt git repository

git clone [url] - kopierer et eksisterende repository fra en URL

git status - viser statusen til filene i repositoryet

git add [filnavn] - legger til en fil til staging area

git commit -m "beskrivelse" - lagrer endringene med en beskrivelse

git push - laster opp endringene til et eksternt repository

git pull - henter og integrerer endringer fra et eksternt repository

git branch - viser alle branchene i repositoryet

git checkout eller switch [Branch-navn] - bytter til en annen Branch

git merge [Branch-navn] - slår sammen begge branchene til 1


Det var noen vanlige git kommandoer,
her kommer resten Github commands:

git log - viser en logg over alle commits i repositoryet

git remote - viser eksterne repositories knyttet til det lokale repositoryet

git fetch - henter endringer fra et eksternt repository uten å integrere dem

git reset [filnavn] - fjerner en fil fra staging area

git diff - viser forskjellene mellom endringer i filene

git stash - lagrer midlertidig endringer som ikke er klare for commit

git tag [tag-navn] - lager en tag for en spesifikk commit

git remote add [navn] [url] - legger til et nytt eksternt repository

git clean - fjerner uønskede filer fra repository

git clear fjerner all teksten fra terminalen

git show - viser detaljer om en spesifikk commit

Git branch -m endrer navnet på git branchen din

git -d (navnet på branchen) sletter en git branch som du har skrevet navnet til
